# Magnifica Humanitas

Encyclical Letter *Magnifica Humanitas* of His Holiness Pope Leo XIV On Safeguarding the Human Person in the Time of Artificial Intelligence.

Markdown versions in [English](markdown/en.md), [Spanish](markdown/es.md), [French](markdown/fr.md), [Italian](markdown/it.md), [Portuguese](markdown/pt.md), [Polish](markdown/pl.md), [German](markdown/de.md) and [Arabic](markdown/ar.md).

A LaTeX version of the English text is available in [latex/en.tex](latex/en.tex), with the compiled PDF in [pdf/en.pdf](pdf/en.pdf) in `pdf/`.

To rebuild the PDF:

```
tectonic latex/en.tex --outdir pdf
```
